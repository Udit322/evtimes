Hey guys, we’ll now use this GitHub repo for the project:

Repo: https://github.com/Udit322/evtimes

⚡ Setup (first time only):

1. Clone repo:
   git clone https://github.com/Udit322/evtimes.git

2. Go inside:
   cd evtimes

3. Switch to dev branch:
   git checkout dev

4. Pull latest:
   git pull origin dev

---

⚒️ Workflow (IMPORTANT):

* Never work on main branch ❌
* Always create your own branch:

  git checkout -b feature/<your-task>

Example:
git checkout -b feature/login

---

💾 While working:
git add .
git commit -m "your message"
git push origin feature/<your-task>

---

🔁 After completing task:

* Create Pull Request on GitHub:
  feature/<your-task> → dev

---

⚠️ Rules:

* Always pull latest dev before starting
* Don’t push directly to main
* Keep commits clean and meaningful

Let’s keep things organized 🚀
